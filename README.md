# Sound-and-Music
## Cristian Ion

### Contents:
- clipped.py : generates a sine wave and clipped sine wave, and then plays them
- chord.py : creates A4,C4,E4,G4 notes, C4 chord, and plays C4,E4,G4, and C4 Chord
